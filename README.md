# Concrete Strength Prediction & Regression Analysis

## Overview
This project analyses a real-world concrete dataset to identify key factors affecting compressive strength and build predictive regression models.

## Objectives
- Perform exploratory data analysis (EDA)
- Identify relationships between variables
- Build and evaluate regression models
- Improve prediction accuracy

## Tools & Technologies
- R
- Linear Regression
- Polynomial Regression
- Data Visualisation

## Key Steps
- Analysed 1000+ observations and 9 variables
- Created correlation matrix and pair plots
- Built simple linear regression model (Water vs Strength)
- Improved model using polynomial regression (up to degree 4)
- Developed multiple regression model with significant predictors
- Performed model diagnostics (residual plots, assumptions)

## Results
- Improved model performance from R² ≈ 0.08 to R² ≈ 0.61
- Identified Cement, Water, and Age as key predictors
- Polynomial regression improved fit but risked overfitting at higher degrees

## Key Learnings
- Importance of feature selection
- Trade-off between model complexity and overfitting
- Validating model assumptions is critical
